My Projects

Office Hours App

•

Spring 2017

The Office Hours app was built as a final project for Computer Science 052 - Full-Stack Web Development. It was built with Tyler Burnam, Clement Nyanhongo, Joshua Utterback, and Sidney Wijngaarde. The result of this project can be found here.

The purpose of this project was to help make TA lab hours more efficient. In many Computer Science classes at Dartmouth, during TA hours, people sit cramped in a stuffy basement room with no windows and write their names in a queue on the blackboard that ends up with arrows pointing all over the place. Another problem that people have with attending office hours is not knowing when they are occurring because the schedule changes at the last second. So, we built a website where students could manage their classes' office hours and review their schedules. Additionally, they can see the publicly posted questions from each TA session and the current queue of people waiting to speak to a TA.

On this project, in addition to coding various parts of both the front-end and the API, I worked in a bit of an organizational position. I set up the continuous integration with Travis as well as the automatic deployments and wrote large sections of the documentation. Additionally, I was the one who submitted the assignments for each of the weekly milestones and made sure that my teammates did their parts.

Changing Perspectives

•

Fall 2016 - Winter 2017

This project was done as part of my work at the DALI Lab. The other developers on the project were Odon (Eddy) Orzsik during the Fall term and Kristie Chow and Alex Chan during the Winter term. The result of this project can be found here.

The purpose of this project was to build an interface for student in K-8 to be able to experience different disabilities. The website is supposed to work in conjunction with the existing curriculum used by Changing Perspectives and found at http://cpne.org/.

During my first term on the project, I built the front-end of the project using React. As the only developer on the project for two terms, during the second term, I worked in a leadership position, overseeing the changes that the new developers were making to the front-end while spearheading the work on building an entire back-end API in ExpressJS with mongoDB.

Mappit

•

Fall 2016 - Winter 2017

This project was done for COSC 098 - Senior Design and Implementation Project as my culminating experience for my major in Computer Science. It was a group project with Sean Cann, Eric Chen, Edrei Chua, Caleb (Junmo) Kim, and Chi Pham. The result of this project can be found here.

The purpose of the project was to create a different system to handle events as many people can have trouble with event information coming in a text format. Mappit solves this problem by presenting the events on a map similar to how airbnb presents their available locations. The map allows people to easily view the events currently occurring near them.

I worked mostly on front-end logic for this project. After helping to set up the main React structure, I also implemented Redux and became the point person of the group on all things React and Redux. I also helped to set up a lot of the styling, including the use of material-ui React components.

Healthmatters

•

Spring 2016

This project was built during my work at the DALI lab at the request of a research group on campus. We built the kiosk on the left which recorded the responses of hundreds of responses a day to a question about students' emotional health.

I joined the project in its final term at Dartmouth. My main contribution was to work on updating the code base so that it can be generalized to more than one question as well as restyling the page. Additionally, I added in integrations with the Giphy API so that, after responding, users were presented with a gif of a cute animal, hopefully incentivizing the use of the kiosk and calming the more stressed users. Finally, I added the functionality to save the results from the daily button clicks and automatically upload them to GitHub at the end of each day.
